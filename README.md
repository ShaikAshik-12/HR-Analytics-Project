# HR-Analytics-Project
The HR Analytics project leverages Excel, Power BI, SQL, MySQL, and Tableau to uncover workforce insights. It features KPIs on attrition, performance, and diversity, with dynamic dashboards, bucket logic, and themed visuals. The project blends advanced analytics with design aesthetics to support strategic HR decision-making.
